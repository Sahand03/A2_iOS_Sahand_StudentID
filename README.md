# ToDo iOS App (Assignment)

## Overview

This project is an iOS To-Do application built using UIKit, Storyboard, and Core Data.

## Features (Planned)

* Add task using text field
* Display tasks in a table view
* Edit tasks using dialog
* Delete tasks using swipe
* Save tasks using Core Data

## Current Progress

* GitHub repository created
* Swift files implemented for controllers and persistence
* Core Data structure started
* Storyboard UI setup initiated
* Basic navigation and structure designed

## Notes

Due to system performance issues (Mac running out of memory while using Xcode and simulator downloads), the full functionality could not be completed.

However, the structure, required technologies, and partial implementation for the assignment have been completed and uploaded.
